WayFarer API

WayFarer is a public bus transportation booking server. Users can register, login, and create a booking. The admins will be in charge of adding all the buses and creating the trips.

This API is constantly updated with new features. If you will like to see a feature, you can send Send A Feature Request.

Validation And Security: Tokens ¶

The endpoints in this API are protected and will require a valid token to access them. This token is generated after a successful User registration. The token will expire after one hour and users will need to generate a new token. This can be done by simply logging in again.

Even with a valid token, some actions, such as, creating and deleting trips and the creation of buses, are protected and will require elevated privileges. When a user signs up, by default they are regular users and will have to be manually approved by an Admin. This is reflected in the token, and the user is prevented to accessing protected routes.
